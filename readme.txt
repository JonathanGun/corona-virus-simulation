Corona-virus-simulator
=======================
Corona-virus-simulator merupakan suatu simulator persebaran corona virus yang memanfaatkan framework .NET Winforms dengan bahasa pemrograman C# serta modul MSAGL untuk menampilkan pola persebaran virus corona.

Anggota :
1. Jovan Karuna Cahyadi / 13518024
2. Jonathan Yudi Gunawan / 13518084
3. William / 13518138

Requirements :
1. Install IDE Visual Studio pada sistem komputer yang anda gunakan
2. Menambahkan / menginstall modul MSAGL pada komputer anda
3. Install MSAGL Graph Viewer pada komputer anda
4. Install MSAGL Drawing pada komputer anda

Usage:
Saat ini hanya mendukung platform Windows
Langkah - langkah:
1. Jalankan Corruption.exe
2. Pilih file konfigurasi populasi,
3. Pilih file konfigurasi peta,
4. Masukkan jumlah angka yang menjadi jumlah hari setelah virus pertama menyebar
5. Simulasikan
6. Tekan tombol reset untuk kembali pada setting awal
